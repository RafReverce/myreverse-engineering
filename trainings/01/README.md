
# 01 - Simple Password Check (CrackMe)

**Цель:** Обойти проверку пароля без знания самого пароля.

## 🛠 Инструменты
- **Compiler:** MSVC (cl.exe)
- **Debugger:** x64dbg (x32 mode)

## 🔍 Анализ и решение
1. **Поиск:** Нашел строку `[-] WRONG! GO AWAY!`.
2. **Логика:** Обнаружил инструкцию сравнения и условный переход.
3. **Патч:** Заменил инструкцию `JNE` на `JE`, тем самым создав обратную логику выполнения.

## 📄 Файлы в папке
- `crackmeOne.cpp` — Исходный код.
- `crackmeOne.exe` — Оригинал.
- `crackmeOneWIN.exe` — Пропатченная версия.
